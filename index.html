건물 에너지 온톨로지 기반 AI Agent

개인 연구진행 중

연구 배경

건축 도면(DWG/DXF)에 포함된 공간 정보는 설비 설계, 에너지 시뮬레이션, 유지관리 등 다양한 단계에서 핵심적인 역할을 합니다. 그러나 이러한 정보는 대부분 비정형 데이터로 존재하며, 활용을 위해서는 수작업 기반의 데이터 추출 과정이 필요합니다.

연구 목표

IFC, BOT, BRICK Schema 등 건축 도메인 온톨로지와 대규모 언어 모델(LLM)을 결합하여, 건축 도면에서 자동으로 공간 정보를 추출하고 자연어 질의응답이 가능한 AI Agent 시스템을 개발합니다.

연구 방법

DWG/DXF 파일에서 레이어 분석 → 선분 세그멘테이션 → Noding → Polygonize → 공간 속성 추출 → JSON 변환 → 자연어 쿼리의 파이프라인을 구축합니다. 온톨로지는 Protégé와 GraphDB를 통해 관리하며, LangChain/LangGraph 기반의 에이전트가 CLIP 멀티모달 임베딩을 활용합니다.

연구 프레임워크

DWG/DXF → Layer Analysis → Segmentation → Noding → Polygonize → JSON → NL Query

CO₂ 히트펌프 급탕기 동적 시뮬레이션

개인 연구

연구 개요

초임계 CO₂ 냉매를 활용한 히트펌프 급탕기(CHPWH)의 Modelica 기반 동적 시뮬레이션 모델을 개발합니다. 가스쿨러, 증발기, 압축기 등 각 컴포넌트의 열역학적 거동을 정밀하게 모사하고, 전체 시스템의 과도 응답 특성과 제어 성능을 검증합니다.

연구 방법

Modelica 기반으로 GasCooler/Evaporator 모델링부터 정상상태 사이클, 동적 모델, 제어 시스템, 전체 통합, 검증까지의 단계별 접근을 수행합니다.

진행 현황

Phase 1–2 — GasCooler/Evaporator 컴포넌트 모델링 완료
Phase 3–4 — 정상상태 사이클 및 동적 모델 개발 진행 중
Phase 5–6 — 전체 시스템 통합 및 실험 데이터 검증 (예정)

엘리베이터 샤프트 에너지 성능 분석

개인 연구

연구 개요

고층 건물 엘리베이터 샤프트의 열적 특성을 분석하고, 굴뚝 효과(stack effect)에 의한 에너지 손실 메커니즘을 규명하여 저감 방안을 제시합니다.

연구 방법

실측 데이터 기반의 열유동 해석과 CFD 시뮬레이션을 통해 고층 건물 내 공기 유동 패턴을 분석하고, 에너지 손실 저감을 위한 설계 대안을 제시합니다.

기대 성과

• 고층 건물 엘리베이터 샤프트의 열손실 정량화 모델 제시
• Stack effect 저감을 위한 설계 가이드라인 도출
• 에너지 절감 효과의 경제성 분석

건물 에너지 소비 패턴 분류 및 이상 탐지

진행 중 · 2024–

Research Overview

건물 에너지 관리 시스템(BEMS)에서 수집된 대규모 에너지 소비 데이터를 기반으로, 비지도 학습(K-Means, DBSCAN 등) 기법을 활용하여 건물 에너지 소비 패턴을 자동 분류합니다. 또한 Isolation Forest, Autoencoder 기반 이상 탐지 모델을 적용하여 비정상적 에너지 소비를 실시간으로 감지하는 프레임워크를 개발하고 있습니다.

Methodology

시계열 에너지 데이터를 특징 추출(Feature Engineering)하여 차원 축소(PCA, t-SNE) 후 클러스터링을 수행합니다. 이상 탐지는 통계적 방법과 딥러닝 기반 방법을 비교 분석하여 건물 유형별 최적 모델을 제안합니다.

Keywords

Clustering · Anomaly Detection · BEMS · Unsupervised Learning · Time Series

딥러닝 기반 건물 에너지 수요 예측

진행 중 · 2024–

Research Overview

LSTM, GRU, Transformer 등 시계열 딥러닝 모델을 활용하여 건물 에너지 수요를 단기(1시간~1일) 및 중기(1주~1개월) 단위로 예측하는 모델을 개발합니다. 기상 데이터, 재실 스케줄, 건물 특성 등 다양한 외부 변수를 결합한 멀티모달 예측 프레임워크를 구축하고 있습니다.

Methodology

Attention 메커니즘 기반의 Seq2Seq 모델과 Temporal Fusion Transformer를 적용하여 다변량 시계열 예측 성능을 비교합니다. 전이 학습(Transfer Learning)을 통해 소규모 데이터 건물에 대한 예측 정확도 향상을 도모합니다.

Keywords

Deep Learning · LSTM · Transformer · Energy Forecasting · Transfer Learning

GIS 공간 데이터와 건물 에너지 시뮬레이션을 연계하여 도시 스케일의 에너지 소비 분포를 분석하고 예측합니다. 건물 유형, 연면적, 준공년도, 기후 데이터 등을 변수로 활용하여 건물군 에너지 수요 모델을 구축하고, 탄소중립 시나리오를 평가합니다.

Methodology

QGIS와 Python 기반 공간 분석 도구를 활용하여 건물군 데이터를 구축하고, 건물별 에너지 시뮬레이션(EnergyPlus)과 통계 모델(회귀분석, Random Forest)을 병행하여 도시 에너지 맵을 생성합니다.

디지털 트윈 기반 건물 에너지 성능 시뮬레이션

진행 중 · 2024–

Research Overview

BIM(Building Information Modeling)의 IFC 데이터를 활용하여 건물 디지털 트윈을 구축하고, 에너지 시뮬레이션을 자동화하는 파이프라인을 개발합니다. 설계 단계부터 운영 단계까지 실시간 에너지 성능 모니터링 및 최적화를 지원합니다.

Methodology

IFC 파싱 및 gbXML 변환을 통해 BIM-to-BEM 자동화 워크플로우를 구축하고, EnergyPlus 연동 시뮬레이션을 수행합니다. IoT 센서 데이터와 시뮬레이션 결과를 결합하여 디지털 트윈 모델을 보정(Calibration)합니다.
